## Score calculation

| Number of cell removed (nb) | Score   |
|-----------------------------|---------|
| nb = 2                      | 5       |
| 2 <= nb < 4                 | nb * 10 |
| 4 <= nb < 6                 | nb * 15 |
| 6 <= nb < 8                 | nb * 20 |
| 8 < nb                      | nb * 30 |
